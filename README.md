Sample web chat room

## Features

### Screens:

* Choose existing user\create new one
* Single chat room
 
### Message types:

* Pain text
* Picture
* External link

### Other

* View chat history

### Tech stack:

* AngularJS
* ASP.Net WebApi
* MSSQL\EF Code first
* RabbitMQ\EasyNetQ
